# Character ASCII Art — The Agent

One robotic AI agent character that evolves across skin tiers.
Simple and blocky at Common. Detailed and epic at Legendary.

The character IS your AI agent — it grows as you do.

---

## Progression Tiers

| Tier       | Aura Cost | Detail Level         |
|------------|-----------|----------------------|
| Common     | 0–100     | Simple blocky bot    |
| Uncommon   | 125–200   | Armored bot          |
| Rare       | 250–350   | Detailed battle bot  |
| Epic       | 400–500   | Elite mech           |
| Legendary  | 500       | Ultimate agent form  |
