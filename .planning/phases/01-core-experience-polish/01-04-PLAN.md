  **2. Insufficient Aura Handling:**
  ```
  ❌ Not enough Aura!
  Need: {price} Aura | Have: {balance} Aura | Short: {difference} Aura

  💡 Tip: Each lesson earns +1 Aura (base). Complete ~{difference} more lessons!
  {If Aura Farmer class: "Your class bonus gives +2 per lesson, so ~{difference/2} lessons!"}

  **3. Class-Locked Item Handling:**
  ```

  ```
  You already own {item_name}!

  {If not currently equipped:}
  Want to equip it? (y/n)
  ```

  **5. Equip vs. Buy distinction:**
  - Buying = spending Aura, gaining ownership
  - Equipping = setting as active (free, requires ownership)
