<task type="auto">
  <name>Task 1: Create idempotent installer script</name>
  <files>install.sh</files>
  <action>
Create install.sh at project root with the following structure:

1. Header: `#!/bin/bash`, `set -e`, `set -u`
2. macOS detection: Exit with message if not darwin
3. Xcode CLT check: If `xcode-select -p` fails, trigger install and exit with re-run instructions
4. Homebrew check: If `command -v brew` fails, install via official script. After install, handle Apple Silicon PATH (`/opt/homebrew/bin` for arm64 - append to ~/.zshrc if not already present, then eval shellenv)
5. Node.js check: If `command -v node` fails, `brew install node`
6. Claude Code check: If `command -v claude` fails, `npm install -g @anthropic-ai/claude-code`. If exists, silent update attempt.
7. Project directory: `mkdir -p ~/Developer/projects`, clone repo if missing, pull updates if exists
8. Completion banner: Show "INSTALLATION COMPLETE" with numbered next-steps (API key from console.anthropic.com, cd to project, run claude, type "start lesson")

Use visual separators (horizontal rules with unicode box-drawing chars) for readability.
Each step should echo status (checkmark for already-installed, package emoji for installing).
Add error handling: if any brew/npm command fails, show friendly error message with manual alternative.

Do NOT include progress.json creation in the installer (Claude handles this on first session per research recommendation "Approach B").
  </action>
