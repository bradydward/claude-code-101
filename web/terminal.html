Ready to do it for real?

Follow these steps to set up Claude Code on your computer:

1

Install Homebrew (Mac package manager):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2

Install Node.js:

brew install node
3

Install Claude Code:

npm install -g @anthropic-ai/claude-code
4

Get your API key from console.anthropic.com

5

Clone the project and start your adventure:

git clone https://github.com/your-repo/claude-code-101.git && cd claude-code-101 && claude

Then type "start lesson" to begin!
